So You're Tellin' Me...?

So You're Tellin' Me...? - Installation Development Project

GitHub

Itch.io

The Project

• A game created for the Installation Development Project module
• The game must use a custom controller powered by an Arduino
• Follow the theme "Tales of Future Past"
• Group project - team size of 3


The Idea

• You play as the greatest chef of all time - literally all time as he's from the future
• You get sent back in time to cook shrimp fried rice for the humans to reduce the shrimp population before the overshrimpulation can take place
• The twist - you are a shrimp...
• Use the custom controller, shaped like a pan, to cook the rice using the right fire level on the hob


My Part

• I was the lead programmer, doing the coding for the game and ensuring the correct information was passed between the Arduino and Unity
• I created the core game mechanics including pan controls, hob controls with the fire and random fire meter, progress bar, ingredients menu, and win/loss conditions
• Used Git to control our changes and work on the project together
